Instruções para executar o projeto

## Requisitos
```
NODE
```

### MONGODB
Nome do banco
```sh
$ use banco_330870
```
Usuário e Senha
```sh
$ db.usuarios.insert({'nome':'admin','senha':'admin'})
```

### NODE
Instalar pacotes - NPM
```sh
$ npm install
```

Executar App
```sh
$ node app.js
```
